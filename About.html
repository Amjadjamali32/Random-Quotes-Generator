About us

What is Random Quotes Generator?

A random quote generator is a software program or online tool that generates random quotations or phrases from a predefined database or collection. These generators typically source quotes from various authors, books, movies, or famous speeches.

How it Works?

It works through the use of an API. A Random Quotes Generator first makes an API request whenever the page is loaded. As you clicks on new quote button it goes to the next quote.

Which Technology is used?

The technoloy used in Random Quotes Generator is tailwind CSS for attractive and responsive layout. The design of this app is created using tailwind CSS the most poweful and famous framework of CSS. Functionality provided in it is totally javaScript. javaScript is used in it to make it functionaly good.

What are its features?

The main features of it:
1. It can generate new Quote (the main functionality).
2. You can copy a quote and paste it where you want.
3. You can send a reponse or review or any suggestion to us through contact form.
Feature which will be implemented in near Future:
